All sorts of tutorials to discoverer!

📖 Following technical tutorials is an excellent way to learn data science at your own pace. Here you will find tutorials and learning exercises to provide you with step-by-step guidance which will help to make complex concepts more accessible.

🚀 Did you know that you can code online? Click on the glasses to get started with Python and R Programming!

✨ In each tutorial, the source code is available and the user is invited to copy and paste scripts into provided online Jupyter programming environment to run. While working with the code, please test it and then hack it by modifying it to behave differently to help you better understand how it works. Do not worry about “breaking the code” because if it does not work, just go back to the tutorial to copy and paste it again.

🍰 Click on the glasses under the menu to open the programming environment and get started in this amazing learning journey!
